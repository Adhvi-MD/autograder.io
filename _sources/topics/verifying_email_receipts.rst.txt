Verifying Submission Email Receipts Manually
============================================
Submission email receipts can be verified by visiting the verification link
in the email or manually using GPG.

First, save the content of the email to a file, say `message.txt.asc`. The
message will look similar to this:

.. code-block:: none

    $ cat message.txt.asc

    The submission's database ID is 42.

    Please visit localhost/web/project/1?current_tab=my_submissions
    to view all available details on these results.

    Mut Suite 1: 2/5

    Suite 1:

Next, download and import Autograder.io's public key. If you're using
a custom deployment of Autograder.io, replace `autograder.io` with your domain
in the url below:

.. code-block:: none

    $ curl -sSL https://autograder.io/api/gpg_public_key/ | gpg --import -

Finally, verify the sender authenticity of the signed message:

.. code-block:: none

    $ gpg --always-trust --verify -r autograderio -e message.txt.asc
