Verifying Submission Email Receipts Manually¶

Submission email receipts can be verified by visiting the verification link in the email or manually using GPG.

First, save the content of the email to a file, say message.txt.asc. The message will look similar to this:

$ cat message.txt.asc
-----BEGIN PGP SIGNED MESSAGE-----
Hash: SHA512

This email contains a summary of your score for
all non-deferred test cases on the submission from jameslp@umich.edu
at 2020-07-15 21:05:35.560848+00:00 UTC for Course 1 Project 1.
The submission's database ID is 42.

Please visit localhost/web/project/1?current_tab=my_submissions
to view all available details on these results.

Mut Suite 1: 2/5

Suite 1:
    Test 1: 1/1
    Test 2: 0/2

Suite 2:
    Test 1: 1/3
    Test 2: 2/2

Total: 6/13

Receipt ID: 567b4ed0851a422f841a25aa38e02871

Next, download and import Autograder.io’s public key. If you’re using a custom deployment of Autograder.io, replace autograder.io with your domain in the url below:

$ curl -sSL https://autograder.io/api/gpg_public_key/ | gpg --import -

Finally, verify the sender authenticity of the signed message:

$ gpg --always-trust --verify -r autograderio -e message.txt.asc
